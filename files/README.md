# Geosierra — Topografía y Geodesia

Sitio web profesional para **Geosierra**, empresa mexicana especializada en topografía, geodesia, vuelos fotogramétricos, estudios hidrológicos y más.

## Sitio en vivo

[https://geosierra.netlify.app](https://geosierra.netlify.app)

## Páginas

| Página | Descripción |
|---|---|
| `index.html` | Landing page con servicios destacados, estadísticas y CTA |
| `servicios.html` | Catálogo completo de 14 servicios con descripciones |
| `nosotros.html` | Misión, visión, valores y equipo de ingenieros |
| `contacto.html` | Formulario de cotización con mapa interactivo |
| `gracias.html` | Confirmación después de enviar formulario |

## Funcionalidades

- **Formulario de cotización** con mapa Leaflet para seleccionar ubicación
- **Envío dual**: WhatsApp (directo) + Netlify Forms (correo electrónico)
- **Mapa interactivo** con Leaflet + OpenStreetMap
- **Anti-spam**: Honeypot + reCAPTCHA de Netlify
- Diseño responsive, tipografía Syne + DM Mono

## Stack técnico

- HTML5 + CSS3 (sin frameworks)
- [Leaflet.js](https://leafletjs.com/) para mapas interactivos
- [Netlify Forms](https://docs.netlify.com/forms/setup/) para envío de formularios
- Hosteado en [Netlify](https://www.netlify.com/)

## Despliegue en Netlify

1. Conecta este repositorio a Netlify
2. Netlify detecta automáticamente el HTML estático
3. Activa **Form notifications** en Dashboard > Forms
4. Agrega notificación por email a: `ing.joelguadarrama@geosierratopografiageodesia.com`

Netlify Forms maneja el backend automáticamente — no se necesita servidor.
